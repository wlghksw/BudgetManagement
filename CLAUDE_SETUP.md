# Claude API 설정 방법

## 1단계: API 키 발급

1. https://console.anthropic.com/ 접속
2. 계정 생성 또는 로그인
3. 왼쪽 메뉴에서 "API Keys" 클릭
4. "Create Key" 버튼 클릭
5. 키 이름 입력 후 생성
6. 생성된 키 복사 (sk-ant-api03-로 시작)

## 2단계: 환경 변수 설정

`backend/.env` 파일을 열고 다음 줄을 추가하세요:

```env
CLAUDE_API_KEY=sk-ant-api03-여기에-실제-키-붙여넣기
```

## 3단계: 서버 재시작

```bash
# 서버 재시작
npm run dev
```

## 확인 방법

CSV 파일을 업로드하면 서버 콘솔에 다음과 같은 로그가 나타납니다:

- AI 분류 성공: `[AI 분류] "스타벅스코리아" -> 식비`
- 키워드 분류: `[카테고리 분류] "스타벅스코리아" -> 식비 (키워드: 스타벅스)`

AI 분류가 작동하면 `[AI 분류]` 로그가 보입니다.
